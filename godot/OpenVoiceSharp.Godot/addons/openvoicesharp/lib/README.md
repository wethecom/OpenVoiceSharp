# Managed and Native Dependencies

Place required managed and native runtime dependencies here (or in your Godot project's preferred plugin runtime path).

Typical managed dependencies:

- `OpenVoiceSharp.dll`
- `OpusDotNet.dll`
- `RNNoise.NET.dll`
- `WebRtcVadSharp.dll`
- `NAudio.Core.dll`
- `NAudio.WinMM.dll` (Windows microphone path)

Typical native dependencies (Windows x64):

- `opus.dll`
- `rnnoise.dll`
- `WebRtcVad.dll`
